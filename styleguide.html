08 · Правила стиля
Border Radius — всегда border-radius: 0. Квадратные углы — фирменный стиль. Без скруглений.
Тёмный фон — сайт работает только на тёмном фоне (#0A0E1A). Светлый режим не предусмотрен.
Шрифты — Orbitron только для UI (заголовки, кнопки, навигация). Exo 2 для всех длинных текстов. Никакие другие шрифты не используются.
Акцентный цвет — только тил-градиент (#1ABCAC → #0D9E90). Никакого оранжевого, красного или других акцентов.
Логотип — минимальный размер 80px по высоте. Только на тёмном фоне. Не растягивать, не менять цвет.
